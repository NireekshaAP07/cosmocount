

? CosmoCount
Track smart. Stay above 75%. Never miss a class again.
CosmoCount is a space-themed, offline-first attendance tracker built for students who want full control over their academic attendance. With a powerful calendar system, event-based absence tracking, and rich analytics, CosmoCount turns attendance management into a clean and visual experience.
